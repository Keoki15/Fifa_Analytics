Just wrapped up my latest individual project — a FIFA Player Performance & Market Value Dashboard ⚽📊
This project was a great opportunity to strengthen my data skills end-to-end: from cleaning and structuring raw data using Excel and SQL, to building a final interactive dashboard that surfaces meaningful insights.
Here are a few key takeaways from the analysis:
• Attacking wide players (RW/LW) and strikers command the highest market values, highlighting how much the modern game prioritizes offensive impact.
• Surprisingly, overall player rating has almost no correlation with market value (0.013), suggesting external factors like age, potential, and marketability play a much bigger role.
• Players aged 21–25 hold the highest value on average — reinforcing how clubs invest heavily in youth and long-term upside.
• Injury proneness showed almost no impact on valuation in this dataset, which opens up interesting conversations about data limitations and real-world context.
